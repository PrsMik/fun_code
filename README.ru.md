# FunLang 

Экспериментальный функциональный язык программирования на Go. Создан на основе книги «Writing An Interpreter In Go», но дополнен поддержкой строгой типизации.

[Read in English](./README.md)

## Основные особенности
- **Интерпретатор**: Быстрое выполнение с поддержкой оптимизации хвостовой рекурсии (TCO).
- **Поддержка LSP**: Автодополнение, переход к определению и подсказки при наведении.
- **Расширение VS Code**: Интеграция с LSP сервером.
- **Форматтер**: Встроенный инструмент для приведения кода к единому стилю.

## Установка
1. Скачайте бинарный файл из раздела [Releases](https://github.com/PrsMik/funlang/releases).
2. Установите расширение VS Code (`.vsix`) из этого репозитория.
3. Укажите путь к интерпретатору в настройках VS Code.

## Пример кода
```funlang
let add: fn(int, int) -> int = fn(a, b) {
    return a + b;
};

let res: int = puts(add(5, 10));
```
